Space Rocks is a new match 3 game on Unity 6 built almost completely with AI.  Join us miners on a voyage across space, gathering exotic minerals, buying tools to mine more quickly, and fuel to get you to the next planet.  Each stone brings more money, and each tool mines more stones. Hear AI generated music and sound effects. See ai generated sprites and background art.  Experience ai-generated match3 game mechanics.  See all AI used in the credits page, and download the code from the opensource github page.
